# 修改xdb的数据头

## 作业要求：

修改xdb日志文件中的klen和vlen，替换4个字节的二进制为10个字节的ascii十进制.
最后结果是klen和vlen共占用20个字节。
长度为12的klen之前存储为"\x0c\x00\x00\x00"，修改后为"0000000012"。

xdb的代码放在仓库的`db`目录下。

> 注意读取的地方要一起更改

之前是：
```
+---------+---------+-----------+-------------+
| klen(4) | vlen(4) | key(klen) | value(vlen) |
+---------+---------+-----------+-------------+

```

更改后是
```
+----------+----------+-----------+-------------+
| klen(10) | vlen(10) | key(klen) | value(vlen) |
+----------+----------+-----------+-------------+

```
